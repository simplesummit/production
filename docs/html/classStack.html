Detailed Description

The hub for all application pages.

Author
Tyler Duckworth
Date
2019
Copyright
BSD 3-Clause "New" or "Revised" License

Stack inherits from QStackedWidget, which allows it to hold multiple widgets simultaneously. The widget on "top" is chosen by selecting its id and is the one visible to the user.

◆ openPage

void Stack::openPage ( bool  _isChanged)
slot

Change the visible page.

Parameters
_isChangedStatus of button. (True is pushed)

The function gets the destination from HomePage and sets that to be the active page.

In the case of the smoke and fluid simulations, it needs to run the StartSim() method to start (or restart) the process of embedding the simulation in the window.
